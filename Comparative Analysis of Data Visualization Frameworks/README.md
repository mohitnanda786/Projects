# Lab Assignment 1: Data Visualization Tools Analysis

This project contains the materials for **Lab Assignment 1**, focusing on the analysis and comparison of modern data visualization tools.

## Course Information
*   **Assignment**: Lab Assignment-1
*   **Topic**: Discuss and analyze different data visualization tools.

## Contents
*   `Lab_Assignment_1.md`: The main detailed report covering Matplotlib, Seaborn, Tableau, and Power BI.
*   `Project_Walkthrough.md`: A step-by-step guide on how to run the code and use the tools.
*   `matplotlib_demo.py`: Python script demonstrating scientific plotting.
*   `seaborn_demo.py`: Python script demonstrating statistical plotting.

## How to Run
1.  **Install Dependencies**:
    ```bash
    pip install matplotlib seaborn pandas numpy
    ```
2.  **Run Demos**:
    ```bash
    python matplotlib_demo.py
    python seaborn_demo.py
    ```
